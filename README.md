# GPS_week_rollover
ПП. Преодоление проблемы временного сдвига при переполнении целочисленного счетчика недель.

## Задачи
- [x] Считывание входного сигнала на одном последовательном порте
- [ ] Корректировка считанных строк
- [x] Отправка данных на другой последовательный порт

## Оборудование
1. DOIT ESP32 Devkit V1
   - Частота до 240 МГц
   - RAM 320 KB
   - FLASH 4 MB
2. GPS модуль LOCOSYS LS23060 v1.0
